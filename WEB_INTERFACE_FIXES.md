## 新增功能详情

### 视频上传测试功能
**位置**: 系统配置页面 (`/config/`)
**功能**:
1. **文件上传**
   - 拖拽或点击选择视频文件
   - 实时上传进度显示
   - 文件格式和大小验证

2. **视频信息提取**
   - 自动获取视频时长
   - 提取视频分辨率
   - 显示帧率信息

3. **文件管理**
   - 生成唯一文件ID
   - 安全的文件存储
   - 访问URL生成

### API接口
- `POST /config/api/upload-video/` - 视频上传
- `POST /config/api/configs/` - 批量更新配置
- `GET /config/api/configs/` - 获取所有配置

## 使用说明

### 访问系统配置
1. 登录系统
2. 点击导航栏的"系统配置"
3. 在配置页面可以:
   - 调整检测参数
   - 设置报警选项
   - 配置视频存储
   - 上传测试视频

### 视频上传测试
1. 在系统配置页面找到"视频上传测试"区域
2. 点击"选择测试视频"或拖拽文件
3. 填写测试描述（可选）
4. 点击"上传并分析"
5. 查看上传进度和结果

### 支持的视频格式
- MP4 (推荐)
- AVI
- MOV
- MKV
- WMV

### 文件限制
- 最大文件大小: 100MB
- 存储位置: `media/test_videos/`

## 技术实现

### 视频信息提取
使用 `ffprobe` 工具提取视频元数据:
- 如果系统安装了 ffmpeg，会自动提取详细信息
- 如果未安装，仍可正常上传，但信息显示为"未知"

### 文件安全
- 生成UUID作为文件名，防止冲突
- 验证文件扩展名，防止恶意文件
- 限制文件大小，防止存储滥用

### 前端交互
- 使用 XMLHttpRequest 实现文件上传
- 实时进度条显示上传状态
- 响应式设计，支持移动设备

## 故障排除

### 如果上传失败
1. 检查文件格式是否支持
2. 确认文件大小不超过100MB
3. 检查网络连接
4. 查看浏览器控制台错误信息

### 如果视频信息显示"未知"
- 这是正常现象，表示系统未安装ffmpeg
- 不影响文件上传和存储功能
- 如需详细信息，可安装ffmpeg工具
