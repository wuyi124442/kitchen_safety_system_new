## 🎯 使用指南

### 启动系统
```bash
# 启动Django服务器
run_django_direct.bat

# 访问Web界面
http://127.0.0.1:8000

# 默认登录账户
用户名: admin
密码: admin123
```

### 主要功能访问路径
- **仪表板**: `/dashboard/` - 系统监控和设置
- **报警历史**: `/alerts/` - 查看和管理报警记录
- **系统配置**: `/config/` - 参数配置和视频上传
- **日志查询**: `/logs/` - 系统日志查看

### 视频上传测试
1. **方式一 - 仪表板快速上传**:
   - 点击仪表板"设置"按钮
   - 在"视频上传测试"区域选择文件
   - 点击"快速上传"

2. **方式二 - 完整配置页面**:
   - 访问系统配置页面
   - 在右下角找到"视频上传测试"
   - 选择文件并填写描述
   - 点击"上传并分析"
