# 演示视频目录

此目录包含用于演示厨房安全检测系统功能的预录制视频文件。

## 视频文件说明

### kitchen_normal_activity.mp4
- **描述**: 正常厨房活动场景
- **时长**: 约60秒
- **包含场景**: 
  - 正常烹饪活动
  - 人员检测演示
  - 灶台监控演示

### fall_detection_scenario.mp4
- **描述**: 跌倒检测演示场景
- **时长**: 约45秒
- **包含场景**:
  - 人员跌倒事件
  - 紧急报警触发

### unattended_stove_scenario.mp4
- **描述**: 无人看管灶台场景
- **时长**: 约90秒
- **包含场景**:
  - 灶台监控
  - 无人看管报警
  - 风险评估演示

### multi_scenario_demo.mp4
- **描述**: 综合场景演示
- **时长**: 约120秒
- **包含场景**:
  - 人员检测
  - 跌倒检测
  - 灶台监控
  - 火焰检测
  - 多种报警类型

## 使用说明

1. 将实际的演示视频文件放置在此目录中
2. 确保文件名与上述列表匹配
3. 视频格式建议使用MP4，编码为H.264
4. 分辨率建议为640x480或1280x720
5. 帧率建议为15-30fps

## 创建演示视频的建议

### 正常活动场景
- 展示人员在厨房中的正常活动
- 包含烹饪、清洁等日常操作
- 确保人员在摄像头视野内清晰可见

### 跌倒检测场景
- 模拟人员跌倒事件（注意安全）
- 展示从正常姿态到跌倒的过程
- 包含跌倒后的状态

### 无人看管场景
- 展示灶台点火后人员离开的情况
- 包含人员返回灶台的场景
- 演示不同的距离和时间情况

### 综合场景
- 结合多种检测场景
- 展示系统的综合检测能力
- 包含多种报警类型的触发

## 注意事项

- 演示视频应该清晰展示系统的检测能力
- 避免过于复杂的场景，确保检测算法能够正确识别
- 视频内容应该安全，避免真实的危险情况
- 可以使用模拟道具来创建安全的演示场景
