# 厨房安全检测系统 - 完整使用指南

## 🚀 快速启动

### 启动Django Web界面
```bash
run_django_direct.bat
```

### 访问地址
- **Web界面**: http://127.0.0.1:8000
- **管理后台**: http://127.0.0.1:8000/admin
- **默认账户**: admin / admin123

**路径**: `/dashboard/`
**功能**:
- 实时系统状态监控

- 活跃摄像头数量

### 3. 系统配置管理
**路径**: `/config/`
**功能**:
- 检测参数调整
- 报警设置配置
- 视频存储设置
- **视频上传测试** (新功能)

### 4. 日志查询
**路径**: `/logs/`
**功能**:
- 系统日志查询
- 实时日志监控
- 日志导出功能

## 🎥 视频上传测试功能

### 功能特点
- ✅ 支持多种视频格式 (MP4, AVI, MOV, MKV, WMV)
- ✅ 文件大小限制 100MB
- ✅ 实时上传进度显示
- ✅ 自动提取视频信息 (时长、分辨率、帧率)
- ✅ 安全文件存储和管理

### 使用步骤
1. **登录系统** - 使用 admin/admin123
2. **进入配置页面** - 点击导航栏"系统配置"
3. **找到上传区域** - 页面右下角"视频上传测试"
4. **选择视频文件** - 点击选择或拖拽文件
5. **填写描述** - 可选，描述测试内容
6. **开始上传** - 点击"上传并分析"
7. **查看结果** - 上传完成后显示文件信息

## ⚙️ 系统配置说明

### 检测设置
- **跌倒检测置信度**: 0.1-1.0，推荐0.8
- **灶台安全距离**: 0.5-5.0米，推荐2.0米
- **无人看管时间**: 10-300秒，推荐30秒

### 报警设置
- **邮件通知**: 启用/禁用邮件报警
- **短信通知**: 启用/禁用短信报警
- **声音报警**: 启用/禁用本地声音
- **报警邮箱**: 接收报警的邮箱地址

### 视频设置
- **保存视频片段**: 是否保存报警时的视频
- **视频片段时长**: 5-60秒，推荐10秒
- **最大存储天数**: 1-365天，推荐30天

## 🔧 故障排除

### 常见问题

#### 1. 无法访问Web界面
**解决方案**:
```bash
# 确保Django服务器正在运行
run_django_direct.bat

# 检查端口是否被占用
netstat -an | findstr :8000
```

#### 2. 报警历史页面空白
**原因**: 数据库中没有报警记录
**解决方案**: 这是正常现象，系统运行后会有数据

#### 3. 视频上传失败
**可能原因**:
- 文件格式不支持
- 文件大小超过100MB
- 网络连接问题

**解决方案**:
- 检查文件格式和大小
- 尝试较小的测试文件
- 检查浏览器控制台错误

#### 4. 配置保存失败
**解决方案**:
- 检查输入值是否在有效范围内
- 确保网络连接正常
- 查看浏览器控制台错误信息

### 日志查看
如果遇到问题，可以查看日志文件：
- Django日志: `logs/django.log`
- 系统日志: 通过Web界面的日志查询功能
